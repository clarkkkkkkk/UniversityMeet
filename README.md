# UniConnect (UniversityMeet)
Implementing like Google Meet and Goole Classroom in ONE PLACE
---

## Features

- Create or join rooms using a unique Room ID  
- Real-time video and audio streaming via WebRTC  
- Peer-to-peer signaling managed with Socket.io  
- Dynamic room management: track users joining and leaving  
- Simple and intuitive interface for classroom-style meetings

---

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind, ShadCN UI 
- **Backend:** Node.js, Express, Socket.io  
- **Real-time Communication:** WebRTC (with Socket.io signaling)  
- **Utilities:** CORS, dotenv for environment variables
